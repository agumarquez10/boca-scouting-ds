# 💙💛 Proyecto Scouting & Hype: Boca Juniors
Sistema de Data Science para identificar fichajes ideales y medir el sentimiento de la hinchada.
Realiza predicciones semanales y publica un tweet con el top 5 fichajes de esa semana.

## 🚀 Tecnologías
* Python (Pandas, Scikit-learn)
* SQL (SQLite)
* NLP (Análisis de sentimiento)
* API de Fútbol
